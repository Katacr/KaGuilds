# ⚙️ Configuration Files

This page details each configuration file in KaGuilds and their options.

---

## 📁 Configuration File Structure

KaGuilds configuration files are located in the `plugins/KaGuilds/` directory:

```
plugins/KaGuilds/
├── config.yml        # Main configuration file
├── levels.yml       # Level system configuration
├── buffs.yml        # Buff system configuration
├── task.yml        # Task system configuration
├── lang/          # Language file directory
│   ├── zh_CN.yml
│   └── en_US.yml
└── gui/           # GUI menu configuration directory
    ├── main_menu.yml
    ├── guilds_list.yml
    └── ...
```
